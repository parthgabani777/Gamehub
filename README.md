# GameHub

An Video library for watching videos
[Live Link](https://gamehub123.netlify.app/)

## Features

-   Landing Page with category filtering
-   Explore Videos
-   Likes Videos
-   Watch later
-   Playlist management
-   History management
-   Login and Signup (Authentication)
